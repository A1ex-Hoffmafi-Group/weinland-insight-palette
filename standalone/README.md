# Ulrich Weinland - Rechtsanwalt Website

Diese Website wurde für VS Code Live Server konvertiert.

## Installation und Verwendung

1. Öffne den `standalone` Ordner in VS Code
2. Installiere die "Live Server" Extension (falls noch nicht installiert)
3. Rechtsklick auf `index.html` → "Open with Live Server"
4. Die Website öffnet sich automatisch im Browser

## Struktur

- `index.html` - Hauptdatei mit allen Inhalten
- `styles.css` - Alle Styles inkl. Dark Mode
- `script.js` - JavaScript für Animationen und Theme Toggle
- Bilder werden aus dem übergeordneten `src/assets/` Ordner geladen

## Features

✅ Dark/Light Mode Toggle
✅ Responsive Design
✅ Scroll-Animationen
✅ Optimierte Performance
✅ Alle Inhalte wie im React-Projekt

## Hinweis

Die Bilder werden aus `../src/assets/` geladen. Stelle sicher, dass der `src` Ordner parallel zum `standalone` Ordner liegt, oder passe die Bildpfade in `index.html` und `styles.css` entsprechend an.
